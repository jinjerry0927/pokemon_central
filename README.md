# Pokemon Champions Guide Web

Pokemon Champions를 위한 한국어 데스크탑 우선 공략/도감/팀빌딩 웹 프로젝트입니다.

## Project Direction

- Desktop-first guide hub for Pokemon Champions players.
- Korean-first content and UX.
- Main value: connect Pokedex, builds, team builder, speed tiers, and damage calculation in one workflow.
- Mobile support is secondary and focused on reading/searching.

## Current Stage

Hosting setup in progress. The app builds as a static export and is ready to
connect to Cloudflare Pages.
